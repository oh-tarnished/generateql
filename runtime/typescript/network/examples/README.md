# @machanirobotics/loom-network examples

Provider examples are split by folder:

- `http/index.ts`
- `graphql/index.ts`
- `websockets/index.ts`
- `svelte/` (tabbed UI demo)

## Run examples

```sh
bun run run:all
bun run run:http
bun run run:graphql
bun run run:websocket
```

## Run Svelte tabs demo

```sh
bun run run:svelte
```

The Svelte page shows request + response payloads for HTTP, GraphQL, and WebSocket in separate tabs.
